FCC File Metadata Microservice

Submit a form object that includes a file upload. When submiting a file, receive the file name, file type, and size in bytes within the JSON response.

Example Usage:

Example Output:


  {
    name: "sea.jpg",
    type: "image/jpeg",
    size: 190307
  }

Example Error:


  {
    statusCode: 400,
    error: "Bad Request",
    message: "File too large"
  }
